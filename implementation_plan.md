# Museum & Exhibition Website

## Plan
Build a stunning, premium museum/art exhibition website with:

### Pages / Sections (Single Page with smooth scroll)
1. **Navigation** - Sticky, transparent-to-dark header with logo
2. **Hero** - Full-screen parallax with exhibition title + CTA
3. **About** - Museum story with image + text split
4. **Current Exhibitions** - Cards with hover effects
5. **Featured Artists** - Grid with portraits and bios
6. **Upcoming Events** - Timeline/list view
7. **Gallery** - Masonry photo grid with lightbox
8. **Tickets / Visit** - CTA section
9. **Newsletter** - Subscription form
10. **Footer** - Links, social icons, contact

### Design Language
- Dark, sophisticated: near-black bg (#0d0d0d), gold accents (#c9a84c)
- Google Font: Playfair Display (headings) + Inter (body)
- Parallax scrolling
- Smooth reveal animations on scroll (Intersection Observer)
- Glassmorphism cards
- Hover micro-animations
- Masonry gallery with lightbox

### Files
- index.html (main)
- css/style.css
- js/main.js
- Generated images for hero, exhibitions, artists, gallery
